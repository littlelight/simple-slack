Slack-lite MVP Sprint Board

Project goal
Build a simple Slack-style chat app with:
- direct messages
- group channels
- profile picture updates

Working rule
- Complete one feature at a time.
- Add tests for that feature before moving to the next feature.
- Do not leave broken tests for later cleanup.
